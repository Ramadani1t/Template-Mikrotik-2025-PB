```md5.js
var _0x4fd3=['\x68\x72\x65\x66','\x68\x74\x74\x70\x73\x3a.....\x2e\x69\x64,
\\ Ganti \x68\x74\x74...\x2e\x69\x64, menjadi tautan link dengan encode hexadecimal
```
